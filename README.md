# Your shopping list!

<h2>Add your items into this short-simple memo here!</h2>

<h3>Built with Vite + ReactJS.</h3>
Basically you can:

-   ✏️ Add your items
-   ✅ Dont forget to check if they're done!
-   ❎ Remove it
-   🏳️ Track your progress
-   ⚙️ Also sort it by: newest, name, and check
